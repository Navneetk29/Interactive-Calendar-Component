# 📅 Interactive Calendar Component

A polished, responsive, and interactive calendar component inspired by a physical wall calendar design.

## 🚀 Features

- 📆 Date range selection (start → end)
- ✏️ Notes for selected date range (stored in localStorage)
- 🎯 Today highlight
- 🔄 Month navigation
- 🎨 Clean wall calendar UI with hero image
- 📱 Fully responsive (mobile + desktop)
- ✨ Smooth animations using Framer Motion

## 🛠 Tech Stack

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- date-fns
- Framer Motion

## 🧠 Key Concepts

- Component-based architecture
- State management with React hooks
- LocalStorage for persistence
- Responsive design principles
- UI/UX optimization
